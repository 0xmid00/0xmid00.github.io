Manual Enumeration (User Information): Link to heading

Windows: Link to heading

  • whoami
  • net user (list all users)
  • net user [username] (get user info)

Linux: Link to heading

  • whoami
  • cat /etc/passwd
  • id

Hostname Enumeration: Link to heading

Common Commands: Link to heading

  • Windows: hostname or systeminfo
  • Linux: uname -a or cat /etc/issue

List Services and Processes: Link to heading

  • Windows: tasklist /SVC
  • Linux: ps aux

Network Enumeration: Link to heading

IP Address: Link to heading

  • Windows: ipconfig
  • Linux: ifconfig

Network Routes: Link to heading

  • Windows: route print
  • Linux: arp -a

Network Connections: Link to heading

  • Windows: netstat -ano
  • Linux: ss -anp

Firewall Enumeration: Link to heading

Windows: Link to heading

  • netsh advfirewall show currentprofile
  • netsh advfirewall firewall show rule name=all

Scheduled Tasks Enumeration: Link to heading

Linux: Link to heading

  • ls -lah /etc/cron*

Windows: Link to heading

  • schtasks /query /fo LIST /v

Enumeration of Applications: Link to heading

Linux: Link to heading

  • dpkg -l or dpkg -l | grep ^ii (installed packages)
  • lsmod (loaded kernel modules)

Windows: Link to heading

  • wmic product get name, version, vendor (installed software)
  • wmic qfe get Caption, Description, HotFixID, InstalledOn (installed updates)

Enumeration of Readable/Writable Files and Directories: Link to heading

Windows: Link to heading

  • Get-ChildItem "C:\Program Files" -Recurse | Get-Acl | Where-Object { $_.AccessToString -match "Everyone\sAllow\sModify" }

Linux: Link to heading

  • find / -writable -type d 2>/dev/null

Enumeration of Unmounted Disks: Link to heading

  • Windows: mountvol
  • Linux: mount

Enumeration of Device Drivers and Kernel Modules: Link to heading

  • Windows: driverquery.exe /v /fo csv | ConvertFrom-CSV | Select-Object 'Display Name', 'Start Mode', 'Path'
  • Linux: lsmod, modinfo <module>

Get Version for a Specific Module (Linux): Link to heading

  • modinfo <module>
